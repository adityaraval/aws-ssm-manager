# Contributing

Thanks for contributing to AWS SSM Manager.

## Setup

```bash
git clone https://github.com/adityaraval/aws-ssm-manager.git
cd aws-ssm-manager
cd app
npm install
```

Run locally:

```bash
npm start
```

## Testing

Run unit tests:

```bash
npm run test:unit
```

Run E2E tests (GUI-capable environment required):

```bash
npm run test:e2e
```

## Pull Requests

1. Create a feature branch from `main`.
2. Keep PRs focused and small.
3. Add/update tests when behavior changes.
4. Update docs (`README.md`, `ROADMAP.md`, etc.) as needed.
5. Ensure CI is green.

## Security

Do not open public issues for security vulnerabilities. Follow `/SECURITY.md`.
